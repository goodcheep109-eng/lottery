# 🎰 台灣大樂透智慧分析系統

自動抓取台灣彩券大樂透歷史開獎資料，進行多維度統計分析，並提供智慧選號功能。

---

## 📦 安裝

```bash
# 1. 安裝 Python 套件
pip install -r requirements.txt
```

---

## 🚀 使用方式

### 第一次使用（匯入全部歷史資料）
```bash
python main.py --update
```
> 會自動從台灣彩券官網抓取 2002 年至今所有開獎紀錄，存到 `data/lotto649.csv`

### 產生分析 Dashboard
```bash
python main.py --analyze
```
> 產生 `dashboard.html`，自動用瀏覽器開啟

### 一鍵更新 + 分析
```bash
python main.py
```

---

## 📊 Dashboard 功能

| 頁籤 | 功能 |
|------|------|
| 📊 總覽 | 統計摘要、全期熱冷號、頻率長條圖 |
| 🔢 頻率分析 | 1-49 號碼出現次數排行、間隔熱力圖 |
| 🌡️ 冷熱號 | 近 30/50/100 期冷熱號分析 |
| 📅 時間分析 | 月份/季度/年度號碼統計 |
| 🔍 規律分析 | 奇偶比、大小號、和值分佈、連號分析 |
| 🎯 智慧選號 | 多條件篩選 + 5種模式推薦號碼 |

---

## ⚙️ 選號模式說明

| 模式 | 說明 |
|------|------|
| 綜合 | 熱號與冷號平衡考量 |
| 熱號優先 | 優先選近期高頻號碼 |
| 冷號優先 | 優先選久未出現號碼 |
| 歷史最強 | 全期出現最多的號碼 |
| 純隨機 | 完全隨機（對照組） |

---

## 🔧 篩選條件

- **奇偶比**：設定奇數號碼個數範圍（建議 2~4）
- **大小比**：設定大號（>25）個數範圍（建議 2~4）
- **和值範圍**：6個號碼加總範圍（歷史平均約 151）
- **連號上限**：最多允許幾組連號
